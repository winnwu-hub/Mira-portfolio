## 📗 案例 ①：Jungle Word Quest · 丛林单词大冒险

### 📌 一句话概述
为 7 岁零基础英语学习者设计的**闯关式词汇学习游戏**，在动物丛林中解锁关卡、收集单词、获得即时反馈。

### 👤 学习者画像
- **年龄**：7 岁（小学低年级）
- **英语水平**：零基础 - 入门
- **学习动机**：外在动机为主（奖励驱动），注意力持续约 10-15 分钟
- **认知特点**：具象思维、喜欢故事和角色、需要即时反馈

### 🎯 学习目标（Bloom's Taxonomy）

| 层级 | 目标 | 实现方式 |
|------|------|---------|
| 记忆 | 识别并说出 10 个动物/自然主题词汇 | 图文配对 + 自动发音 |
| 理解 | 将词汇与对应的动物/自然概念建立关联 | 情境化图片 + 音效 |
| 应用 | 在游戏中正确选择对应词汇 | 闯关机制中的选择题 |

### 🎮 教学设计亮点

- **🕹️ 3 关 × 逐步解锁**：第 1 关 3 词 → 第 2 关 3 词 → 第 3 关 4 词，难度递进、成就感递增
- **🔊 自动发音 + 手动重播**：每个词汇进入画面自动朗读，配有 🔈 按钮可反复收听，建立音-形-义连接
- **🎵 沉浸式氛围**：背景音乐 + 通关音效 + 动物叫声，降低学习焦虑
- **✅ 即时正误反馈**：选择正确 → 绿色 + 庆祝音效；错误 → 红色 + 鼓励重试

### 🛠️ 技术实现
- 纯 HTML/CSS/JS 单文件，离线可用
- Web Speech API + Web Audio API 双通道音频
- 事件委托架构，可扩展至更多关卡
- 用户手势驱动的音频授权流程

## 📘 案例 ②：My School Life · 我的校园生活

### 📌 一句话概述
面向 12 岁小升初学习者的 **8 课时完整单元自学课件**，以学校日常生活为主线，融合任务型教学法（TBLT）。

### 👤 学习者画像
- **年龄**：12 岁（4-5 年级 / 小升初）
- **英语水平**：A1-A2（初级）
- **学习模式**：自定节奏的自主学习
- **学习动机**：工具型动机（升学需求）+ 内在动机（话题贴近生活）

### 📚 单元结构（8 课时 + 螺旋递进）

| 课时 | 主题 | 核心内容 | 练习类型 |
|------|------|---------|---------|
| 1 | Subjects | 学科词汇 + 表达偏好 | 选词填空 |
| 2 | Timetable | 时间表达 + 课程表 | 选词 + 填空 |
| 3 | Routine | 日常活动 + 时间顺序 | 配对 + 选择 |
| 4 | Classroom Language | 课堂用语 + 请求表达 | 分类 + 配对 |
| 5 | Places | 校园地点 + 方位介词 | 选词填空 |
| 6 | Rules | 校规表达 + 情态动词 | 填空 + 分类 |
| 7 | Friends | 人物描述 + 交友对话 | 配对 + 综合 |
| 8 | Review Challenge | 全单元综合回顾 | **综合挑战** |

### 🎯 教学设计亮点

- **🌀 螺旋递进路径**：词汇 → 短语 → 句型 → 对话 → 综合运用，每课在前一课基础上扩展
- **📝 4 种练习题型**：选词填空 / 配对连线 / 分类归类 / 综合挑战，适配不同学习风格
- **⭐ 自定节奏 + 评分系统**：每课独立评分，学习者按自身进度推进
- **🔊 词汇发音集成**：所有新词汇支持点读，建立音形连接
- **🎯 TBLT 任务驱动**：每课围绕一个真实交际任务（如「用英语介绍你的课程表」）

### 🛠️ 技术实现
- 纯 HTML/CSS/JS 单文件
- 多题型练习引擎（mc/fill/match/sort）
- 8 课 × 4 题型 = 可复用的组件化架构
- 评分系统 + 课程解锁机制

## 📙 案例 ③：BizTrip English · 商务出差英语

### 📌 一句话概述
面向成人职场英语学习者的 **Storyline 风格故事驱动交互式课件**，以「北京→新加坡商务出差」为主线，在三个真实任务中训练**语域切换能力**。

### 👤 学习者画像
- **年龄**：22-40 岁在职成人
- **英语水平**：A0-B1（课件内部三级难度自适应）
- **学习动机**：工具型动机（职业发展）+ 融合型动机（国际商务文化）
- **核心痛点**：不会根据场合切换表达方式（语域意识薄弱）

### 🗺️ 故事主线

```
✈️ 机场出行 ──🔒──→ 🤝 客户会议 ──🔒──→ 🍷 晚宴社交
事务型语域          正式商务语域          非正式社交语域
```

### 🎯 学习目标（Bloom's Taxonomy 全覆盖）

| 层级 | 目标 | 实现环节 |
|------|------|---------|
| 记忆 | 识别 20+ 商务旅行核心词汇 | 📚 Prep 词汇配对练习 |
| 理解 | 理解航班延误通知、会议议程、餐厅推荐 | 🎯 Mission 听懂 NPC 对话 |
| 应用 | 在模拟场景中作出符合交际目标的回复 | 🎯 Mission 对话交互 |
| 分析 | 识别自己用语的语域是否与场合匹配 | 📊 Debrief 语域分析 |
| 评价 | 对比三个场景语域差异，反思表达选择 | 📊 Debrief + 跨场景对比 |

### 🔬 核心教学设计

#### 🎚️ 三级难度实时切换

| | 🟢 引导模式 (A0-A1) | 🟡 半自由模式 (A1-A2) | 🔴 自由模式 (A2+) |
|---|---|---|---|
| 输入方式 | 点击完整句子选项 | 输入框 + 关键词提示 | 纯空白输入框 |
| 纠错反馈 | 选错 → 即时显示正确句 + 解释 | 提交后 Recast 纠错 | 仅 Debrief 总结性反馈 |
| 脚手架 | 完整选项 + 关键词高亮 | 2-3 个关键词提示 | 0 提示 |

#### 🪜 脚手架渐进撤除（贯穿 3 个任务）

即使全程使用 🟢 引导模式，选项提示也在逐步减少——**不被察觉的难度递进**。

#### 🗣️ 语域维度（独特评分创新）

| 任务 | 语域类型 | 期望特征 |
|------|---------|---------|
| ✈️ 机场 | 事务型 | 简短、直接、信息交换优先 |
| 🤝 会议 | 正式商务 | 结构化、礼貌、关系+内容并重 |
| 🍷 晚宴 | 非正式社交 | 轻松、自然、关系建立优先 |

### 📊 评估体系

| 维度 | 权重 | 说明 |
|------|------|------|
| 🎯 任务完成度 | 40% | 是否完成每轮交际目标 |
| 📝 语言准确度 | 30% | 词汇/句型正确性 |
| 🗣️ 语域适当性 | 30% | ⭐ 场合感匹配（本课件独有） |

### 🧠 教学理论支撑

| 理论 | 在课件中的体现 |
|------|--------------|
| TBLT（任务型教学法） | 三个交际任务 × 明确产出目标 |
| Krashen i+1 | AI 输入略高于学习者当前层级 |
| Swain Comprehensible Output | 🔴 自由模式强制产出 + 延迟反馈 |
| Vygotsky ZPD | 三级难度对应三个最近发展区 |
| Knowles Andragogy | 立即有用的职场内容 + 自我导向 |
| Lyster & Ranta | Recast + Elicitation 纠错策略 |
| SDT（自我决定理论） | 自主选难度 + 评分反馈 + 故事归属感 |

### 🛠️ 技术实现
- 纯 HTML/CSS/JS 单文件（1570 行）
- JSON 数据驱动架构：修改/新增场景只需编辑数据数组
- Fisher-Yates 选项随机化算法
- 事件委托 + 屏幕状态机
- localStorage 进度持久化
- Web Speech API TTS（自动过滤舞台指示词）
