Interactive courseware portfolio for Learning Designer roles at AI/tool-based online education companies.

**[View Portfolio →](https://winnwu-hub.github.io/ld-portfolio/)**

## 📂 Cases

| # | Case | Audience | Type | Key Feature |
|---|------|----------|------|-------------|
| ① | Jungle Word Quest | Ages 7 | Vocabulary Game | Auto-pronunciation + unlockable levels |
| ② | My School Life | Ages 12 | Self-Study Unit | 8 lessons × 4 exercise types |
| ③ | BizTrip English | Adults | Story-Driven Simulation | 3-tier adaptive difficulty + register scoring |

## 🛠️ Tech Stack

All cases are built as **single-file HTML/CSS/JS** prototypes — no dependencies, no build step, open in any browser.

- Web Speech API (TTS pronunciation)
- Web Audio API (sound effects)
- Event delegation architecture
- Data-driven rendering (Case 3)
- localStorage persistence
- Fisher-Yates randomization
